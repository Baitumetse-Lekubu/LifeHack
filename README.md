# 📱 QuizMe – Life Hack or Urban Myth?

## 🧩 Project Overview
**QuizMe** is a fun and interactive flashcard application developed as part of the **IMAD5112 Assignment 2**.  
The app challenges users to distinguish between genuine real-life hacks and viral urban myths.

By testing their knowledge, users learn practical tips to improve everyday decision-making while tracking their performance through a built-in scoring system.

---

## ✨ App Features
- 🎯 **Top-Branded Header**  
  Clean and clear *"QuizMe"* branding at the top of the interface.

- 🃏 **Interactive Flashcards**  
  Includes 5 unique questions covering common household tips and myths.

- ⚡ **Immediate Feedback**  
  Real-time results and explanations after each answer.

- 📊 **Score Tracking**  
  Automatic scoring system with results calculated out of 5.

- 🏆 **Personalized Ranking**  
  Dynamic feedback based on performance:
  - *"Master Hacker!"*
  - *"Stay Safe Online!"*

- 🔍 **Review System**  
  Users can review all correct answers at the end of the quiz.

---

## 🎯 Target Audience
This app is designed for:

Anyone interested in improving daily efficiency and digital literacy by debunking common myths will benefit from using QuizMe.

---

## 🚀 How to Use the App

### 1. Welcome Screen
- Read the app description to understand the quiz objective.
- Click **Start** to begin.

### 2. Flashcard Questions
- Read the statement displayed on the screen.
- Select:
  - **Hack** → if you think it’s true  
  - **Myth** → if you think it’s false  
- Review the explanation provided.
- Click **Next Question** to continue.

### 3. Score Screen
- View your final score out of 5.
- Read your personalized performance message.
- Click **Review** to see all correct answers.
